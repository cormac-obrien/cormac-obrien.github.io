GSoC 2015

Adding Support for Mac OS 9 to QEMU

Overview

As of version 2.2.1, QEMU does not support emulation of a machine running Mac OS 9. This GSoC project aims to implement support for OS 9.

Status

2015.03.27: The out-of-memory error occurs in bootinfo_init_program while trying to allocate memory to load the boot script. The script is only about 80 lines long, so it seems like the load-size word isn't working properly.

2015.03.25: OpenBIOS fails to boot from the OS 9.2.2 install ISO. This appears to be the result of an out-of-memory error. The launch command given is 

$ qemu-system-ppc -m 256 -cdrom os9.iso -boot d
